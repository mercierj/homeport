## Definition Of Full

Every service row must pass this checklist:

- `discover`: Terraform/API parser finds the source resource and dependencies.
- `cost`: HomePort compares current and target costs.
- `provision`: HomePort creates the target service.
- `migrate`: HomePort moves data/config/state.
- `api_compat`: existing SDK/protocol keeps working, or the row is marked impossible with reason.
- `env_dns`: HomePort injects env vars, endpoints, aliases, certs, or DNS changes.
- `ha`: HA/failover behavior is reproduced or explicitly downgraded with user approval.
- `backup`: backup and restore are configured.
- `validate`: HomePort runs conformance checks.
- `cutover`: HomePort switches traffic/config.
- `rollback`: HomePort can revert.
